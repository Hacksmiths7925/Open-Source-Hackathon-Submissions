## Project Overview
Mediscan AI Analysis PRO is a smart generative healthcare framework that utilizes OCR (Optical Character Recognition) to extract and analyze data from medical reports, providing instant healthcare insights and assistance.

## Repository Link
- **GitHub Repository:** https://github.com/Hacksmiths7925/Mediscan-analysis-AI-

## Key Features
- **Medical Report Image Upload:** Seamlessly upload medical reports in image format.
- **OCR Text Extraction:** Accurately extracts text data from medical documents.
- **Health Parameter Detection:** Identifies critical health parameters and markers.
- **Medicine Reminder:** Interactive reminders for prescribed medications.
- **Nearby Hospital Locator:** Quick utility to find healthcare facilities.

## Tech Stack
- **Language:** Python
- **Framework/UI:** Streamlit
